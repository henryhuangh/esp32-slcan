# ESP32 SLCAN Firmware

This project uses FreeRTOS to handle CAN bus data to send and receive commands with the SLCAN protocol.

## SLCAN Protocol

The manual for the SLCAN protocol can be found at [this link](https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://www.canusb.com/files/canusb_manual.pdf&ved=2ahUKEwju_9zlwMuVAxVPNYYAHcdrExoQFnoECBgQAQ&usg=AOvVaw0kr_4KbPpo8GMB-Emn3ZYC)
